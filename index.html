Jawbone for SARAH

Documentation plugin Jawbone for SARAH

Pré-requis

Pour utiliser le plugin, il vous faut récupérer certaines infos qui vous sont personnelles sur le site de Jawbone.

Inscription sur le site de Jawbone

Commencez par aller vous inscrire sur le site de Jawbone : https://jawbone.com/up/developer/.
Il vous faut ensuite créer une application sur ce même site (avec des infos plus ou moins bidon).
N'oubliez pas de remplir le champ "OAuth redirect URLs", c'est nécessaire pour récupérer un token d'authentification.
Une fois la création de votre app terminée, notez les infos suivantes : Client Id et App Secret ( = client secret).

Récuperer l'access token

Accéder à l'adresse suivante via votre navigateur : https://jawbone.com/auth/oauth2/auth?response_type=code&client_id=xxx&scope=basic_read%20extended_read%20friends_read%20move_read%20sleep_read&redirect_uri=https://test
Attention pensez à remplacer dans l'url ci-dessus le client_id par le votre et le redirect_uri par l'adresse que vous avez mis dans le champ "OAuth redirect URLs".
Si tout va bien, vous devez voir une page qui vous demande d'autoriser l'application à accèder à vos données. Acceptez et vous allez être redirigé vers la page que vous avez indiquée dans le champ "OAuth redirect URLs".
Notez le paramètre "code" qui est dans l'url.
Ensuite saisissez l'url suivante : https://jawbone.com/auth/oauth2/token?client_id=xxx&client_secret=yyy&grant_type=authorization_code&code=zzz.
Pensez à remplacer les xxx, yyy et zzz par vos valeurs.
Cette page doit vous affichez du Json. Copiez la valeur du access_token. Pour info, il est valable 1 an.

Paramètrer le plugin

Copier les valeurs access_token et client_secret dans le fichier jawboneup.prop.
